Sample #14 for yaneSDK2nd

This demonstrates real-time fading with the yga format (a reversible compression format with alpha channel),
and lens flare expression using universal transition.

The image data is from WAFFLE's "HAPPY Hotarusou".
It will be released at the end of the year, please consider purchasing it ^^;

(However, data/light.jpg can be freely used)

Regarding the source code, you are free to quote and use it.
To compile the source code, yaneSDK2nd version 1.54 or later is required.
