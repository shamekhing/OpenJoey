Sample #18 for yaneSDK2nd

A sample demonstrating stencil alpha and silhouette alpha
for the yga format (a compressed image format with anti-aliasing information).

The yga image data is from the title screen of WAFFLE brand's "Aoki Daichi".
(Scheduled for release in March 2001, please consider purchasing it ^^;)
The bg data is from the title screen of WAFFLE brand's "HAPPY Hotarusou".

While redistribution of this archive itself is permitted, please
refrain from secondary use of the images.

Regarding the source code, you are free to quote and use it.
To compile the source code, yaneSDK2nd version 1.56 or later is required.
