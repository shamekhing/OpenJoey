
layout_define.txt	:	画面レイアウトを設定ファイル
bg_define.txt		:	BG(背景)のファイル名を登録します
se_define.txt		:	SE(効果音)のファイル名を登録します
bgm_define.txt		:	BGM(音楽)のファイル名を登録します
sc_define.txt		:	StandChara(立ちキャラ)のファイル名を登録します
face_define.txt		:	顔マークのファイル名を登録をします
name_define.txt		:	ネームプレートのファイル名を登録します

	☆　ファイル名の登録方法

	０番から順番にファイル名を登録していきます。
	シナリオファイル側からは、この番号で呼び出します。
