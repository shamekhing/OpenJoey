yaneSDK2nd用YGA maker

PhotoShopで透明部分を黒にした画像と白にした画像の２つの画像から、
αチャンネル付き可逆圧縮形式(yga形式)のデータを作ります。

詳しくは、やねうらおのホームページ
	http://www.sun-inet.or.jp/~yaneurao/
の「スーパープログラマーへの道　第CB回　PhotoShopのレイヤ合成
アルゴリズムについて（YGAフォーマットを提唱する) 00/10/18」を
ご覧ください。

「黒画像」と「白画像」の作成はPhotoShopならば、レイヤを一枚新規で
作成し、それを一番後ろに置き、そのレイヤを黒でペイントして、画像
全体をbmpで保存したものが黒画像。白でペイントして、画像全体を
bmpで保存したものが白画像です。

ソースに関しては、自由に引用してご利用ください。
ソースのコンパイルには、yaneSDK2nd 1.00β9以降が必要です。

実際にYGA形式の読み込み、転送等には、yaneSDK2nd 1.00β2以降が
必要です。

また、この実行ファイルでyga画像の確認もできます。
（１秒間、背景が黒から白になりながら、その画像が見えます）
