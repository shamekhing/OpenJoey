template <class T> class vectorEx　【テンプレート】

template<class T> class chain : public vectorEx<T*>【テンプレート】

☆　内容

std::setには、重複を許さない要素追加を行なう、insert，要素の削除を行なうeraseという関数があるが、これをstd::vectorに追加したのが、このvectorExである。

setのinsert,eraseは速いが、その半面、iteratorで巡回するのが遅い。insert,eraseは頻繁に行なわないが、iteratorで巡回するのは速くあって欲しい場合は、setは適さない。しかしvectorだと、insert,eraseをいちいち記述するのは面倒である。そこで、insert,erase機能つきのvectorを作ったわけである。

また、chainテンプレートは、単なるvectorEx<T*>であって、これは、ポインタ列を扱うのに使う。（と便利かも知れない）

☆　使いかた

pair<iterator,bool> insert(T t);

set的追加（同一要素の存在を認めない）
set::insertと同機能 /insert出来たとき時に<it,true>
insert出来なかった時は<同一要素へのイテレータit,false>

bool erase(T t);

// set的削除(同一要素の存在を削除する)
// set::eraseと同機能/削除した時にtrue
