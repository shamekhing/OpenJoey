class CTimeBase

☆　内容

ミリ秒単位の時間を取得するためのベースクラスです。
環境に応じてWin32のtimeGetTimeとGetTickCountとを呼び分けます。

☆　メンバ関数

static DWORD timeGetTime(void);　仕様はWin32のtimeGetTimeと同じです。

補助関数

static int timeBeginPeriodMin(void); // タイマを最小設定にする（返し値 interval[ms]）
static void timeEndPeriodMin(void); // タイマ最小設定解除

☆　注意点

コンストラクタでこのクラスを初期化するので、使用に際しては、どこかにこのクラスのインスタンスが存在する必要があります。
（ディフォルトではyaneTimer.cppの中にある）
