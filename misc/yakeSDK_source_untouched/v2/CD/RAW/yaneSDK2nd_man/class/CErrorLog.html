class CErrorLog　【コンポーネント】

☆　内容

エラーログの出力のための基底クラス。
使用に際しては、yaneConfig.hでUSE_ErrorLogがdefineされている必要があります。（ディフォルトでdefineされています）

☆　メンバ関数

virtual LRESULT Out(string); // 文字列出力
virtual LRESULT Out(int); 　　// 数値の出力
virtual LRESULT __cdecl Out(LPSTR fmt, ... ); // printfの書式出力
virtual LRESULT SelectDevice(int); // エラーログ出力先。(0:なし 1:File　ファイル名は"ErrorLog.txt")
virtual string GetError(void) const; // 前回エラー出力したものを文字列として取得する。

エラーを出力すると、そこにはそのときの日付と時刻も同時に出力される。

☆　注意

出力先は、DEBUGビルドの時は、ディフォルトでファイル(SelectDeviceで1を指定した状態)。
非デバッグモードのときは、ディフォルトで、なし(SelectDeviceで0を指定した状態)。

☆　グローバル変数

このクラスのグローバル変数としてErrというのを用意している。(yaneError.cpp内)
通常は、この変数に対してエラーを出力すれば良い。

例）
　Err.Out("This may be a bug...");
