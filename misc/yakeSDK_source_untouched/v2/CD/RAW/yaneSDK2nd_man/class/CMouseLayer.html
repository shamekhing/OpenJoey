class CMouseLayer【コンポーネント】

☆　内容

ソフトウェアマウスカーソルです。DirectDrawを使うと、画面描画に際してマウスカーソルのちらつきが発生するので、それをソフトウェアで描画する手段を提供します。CPlaneBaseを渡すので、CDIB32,CDirectDrawのどちらでも動作します。

☆　メンバ関数

CMouseLayer(smart_ptr<CPlaneBase>v,int x=0,int y=0);
CMouseLayer(void);

コンストラクタは２通りある。前者は、プレーンのポインタと、プレーンのどの位置がマウスカーソル先端部なのかというオフセットを指定する。前者は、即、使用状態になる。後者は、そのままでは使用しない状態になるので、この後でSetPlaneしなければ使えるようにはならない。

void Enable(bool bEnable); // 有効にする
bool IsEnable(void); // 有効か？

void SetPlane(smart_ptr<CPlaneBase>v,int x=0,int y=0);

コンストラクタCMouseLayer(void)の、あとでプレーンを設定するのに使う。SetPlaneの各パラメータはコンストラクタで指定しているものと同じです。
