class CSinTable 【コンポーネント】

☆　内容

sin,cosのテーブルを提供します。高速なsin,cosが実現できます。

☆　メンバ関数

// n:0-511で一周（360゜）。結果は<<16されて返る
static LONG Cos(int n){ return m_lTable[n & 511]; }
static LONG Sin(int n){ return m_lTable[(n+128) & 511]; }

テーブルは、このクラスのコンストラクタで初期化するため、CSinTableクラスを使用する場合は必ずどこかでコンストラクトする必要がある。

例）
sin,cosを使用したい関数のあるcppファイル内ではすべて、
　　static CSinTable gSinTable;
のようにして、gSinTable.Cos(x)だとかgSinTable.Sin(y)だとかして使うと良いです。
