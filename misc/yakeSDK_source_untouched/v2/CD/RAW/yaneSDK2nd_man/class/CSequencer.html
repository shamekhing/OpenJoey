class CSequencer　【コンポーネント】

☆　内容

タイミングの制御用。時間に合わせたアニメーション等を実現するときに使うためのアシストクラス。
ある範囲start≦x＜endに対して、値dataを対応づける関数Add，ある位置における値の対応を調べる関数Getから成る。

☆　メンバ関数

LRESULT Add(int start,int end,int data); // 追加
LRESULT Add(int start,int data); // 連続追加するときに使う。上のendは省略できるバージョン。
LRESULT Del(int start,int end,int data); // 削除

void Clear(void); // 全削除

LRESULT Get(int pos,int& data,int &diff);
// posの値に対応するデータについて問い合わせる。
// 見つかったときにはdataにその値、diffにstartからの増量が入る。

☆　使用例

たとえば、アニメーション開始後、1000[ms]〜2000[ms]の間は、１番のグラフィック、2000[ms]〜3500[ms]の間は、２番のグラフィックを表示したいとすれば、

　CSequencer seq;
　seq.Add(1000,2000,1);
　seq.Add(2000,3500,2);
として、そのあとタイマー(CTimerを参照のこと)を見ながら

　int nSceneNo = seq.Get(timer.Get() , data , diff ));

とやれば、1000〜2000[ms]の間は、１，2000〜3500[ms]の間は２が返ってきます。
