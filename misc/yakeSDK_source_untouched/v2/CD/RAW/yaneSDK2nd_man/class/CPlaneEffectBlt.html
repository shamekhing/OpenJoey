class CPlaneTransBlt

☆　内容

CPlaneBaseに対するトランジション関数です。CPlaneBaseは、CPlane,CDIB32の共通基底クラスなので、このどちらに対してもトランジションが行なえます。

トランジションとは、初期状態（何も表示されていない）から、あるグラフィックの完成状態へ、なめらかに遷移するための表現です。
トランジション関数では、フェーズを指定します。初期状態は０，完成状態は２５６で、この間の数字が、その中間というわけです。

☆　メンバ関数

このクラスのメンバ関数は、すべてstaticで、以下のプロトタイプによる。

#define TransBltFunc(FuncName) \
LRESULT FuncName(CPlaneBase*lpDst,CPlaneBase*lpSrc,int x,int y,int nPhase,int nTransMode = 0);

lpDstは、転送先のCPlaneBaseのポインタ。もちろんCPlane,CDIB32のポインタを渡せば、暗黙のアップキャストによって、うまく関数が呼び出される。lpSrcは転送元のCPlaneBaseのポインタ。x,yは表示座標。nPhaseは、トランジションのフェーズ（０−２５６） nTransModeは、ヌキ色を有効にするかどうか。nTransMode : 0 == ヌキなし 1==ヌキ有り 2==α付き画像(CDIB32限定)　3==BltNaturalでの転送です。2を指定すれば、α値付きの画像に対してもトランジションが可能になります。3ならば、CPlane,CDIB32,CDIB32のα値付きの画像すべてに対してうまく動作するトランジションとなります。（CPlaneBase::BltNaturalについても参照のこと）
