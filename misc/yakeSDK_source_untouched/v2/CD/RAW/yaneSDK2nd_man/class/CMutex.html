class CMutex　【コンポーネント】

☆　内容

Mutexクラスです。ＯＳレベルで１つしか存在しない相互排他を目的とします。２重起動の防止などに使います。
２重起動防止は、CSingleAppも参照すること。

☆　メンバ関数

LRESULT Open(string szMutexName);

　szMutexNameは、ミューテックス名（任意につけて良い）

　返し値は
　0 : 他に同名のものは存在しない→所有権獲得成功！
　1 : 他に同名のものが存在する。(error)

void Wait(void); // 所有権が獲得できるまで待つ

　Openで１が返ってきているときに呼び出して、所有権が獲得できるのを待つ。

void Release(void); // Mutexを解放する

　デストラクタも自動的にこのReleaseを呼び出すので、明示的に呼び出す必要は無い。

class CMutexSection　【コンポーネント】

☆　内容

MutexによるLock〜unlockです。CCriticalSectionと同じ感覚で使えます。

☆　メンバ関数

void Enter(string szMutexName); // MutexSectionに入る
void Leave(); // MutexSectionから出る

// デストラクタではCMutexを解放するので、そのときに
// 自動的にMutexは解放される。
