class CCellAutomaton

☆　内容

セルオートマトンクラスです。

セルオートマトンとは、ピクセル１つ１つが何らかの規則を持っていて、その規則に基づいて次の状態に移り変わっていくものです。
たとえば、上にスクロールしながらぼかすならば、

＊□＊
■＊■

□の次の瞬間のピクセル値(RGB)は、その左下と右下の２つ■のピクセル値(RGB)の加重平均で決まるとします。

いろいろ応用は出来そうです。面白い画面効果が出来たら、やねうらおまでメールで教えてください＾＾

☆　メンバ関数

CDIB32のポインタを取り、lpRectで指定された矩形に対して処理を行ないます。lpRect==NULLの場合は、プレーン全域となります。

static void UpFade(CDIB32* lpSrc,LPRECT lpRect=NULL); 　　// 上に流れながらフェード
static void DownFade(CDIB32* lpSrc,LPRECT lpRect=NULL); // 下に流れながらフェード
static void LeftFade(CDIB32* lpSrc,LPRECT lpRect=NULL);　 // 左に流れながらフェード
static void RightFade(CDIB32* lpSrc,LPRECT lpRect=NULL); // 右に流れながらフェード
