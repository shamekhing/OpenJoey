class CFPSLayer【CHDCLayer派生クラス】

☆　内容

ＦＰＳ（Frames Per Second／１秒当たりのフレーム数）を描画するレイヤです。CHDCLayerから派生させています。

☆　メンバ関数

virtual void OnDraw(HDC hdc); // これはCHDCLayerからローバーロードしたもの
CFPSLayer(CFPSTimer*); // コンストラクタで描画に使っているCFPSTimerへのポインタを渡す。

使いかたは、いたって簡単。CFPSLayerをコンストラクタで渡せば、あとはCHDCLayerなので自動的にCDirectDraw::OnDrawでコールバックされます。

void SetSkipFrameDraw(bool bEnable); // スキップしたフレーム数も描画する(ディフォルトではfalse)
スキップしたフレーム数については、CFPSTimer::ToBeSkipメンバ関数を参照のこと。
