class CInteriorCounter　【コンポーネント】

☆　内容

内分カウンタを提供します。カウンタとしてはCRootCounterなども参考にしてください。

Setメンバ関数で、初期値(nStart)・終了値(nEnd)、そして分割数(nFrames)を設定します。そうすれば、このカウンタはnStartから始まり、nFrame回のIncメンバ関数の呼び出しによってnEndに到達するような内分カウンタとなります。

また、Incは、operator++として定義してあるので、++とも書けます。
Incの逆操作。Decもあり、それはoperator--として定義してあり、--とも書けます。
