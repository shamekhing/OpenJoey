class CAppIntervalTimer

☆　内容

CAppBaseに寄生するインターバルタイマー。使いかたは、CIntervalTimerと同じく、

virtual void TimerProc(void); // コールバックされる関数

を継承して使う。こちらは、CIntervalTimerと違い、システムリソースを食わないので、地球に優しい＾＾；

CVolumeFaderで使っているので、そちらも参考にしてください。

☆　メンバ関数

LRESULT Start();

　インターバル割り込みを開始する。そうすると仮想関数TimerProcをコールバックする。コールバックは、CAppBase::IsThreadValidのなかから行なわれる。

LRESULT Stop(void); // タイマを止める

　Startでスタートさせたインターバル割り込みを停止させる。

bool IsTimer(void) const; // タイマが回っているのか

　Startでスタートさせたあとは、Stopするまで、この関数はtrueを返す。
