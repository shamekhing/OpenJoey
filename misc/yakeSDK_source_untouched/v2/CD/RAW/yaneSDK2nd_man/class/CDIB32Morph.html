class CDIB32Morph

☆　内容

　モーフィングの手助けを行ないます。モーフィングとは、ソース１の画像からソース２の画像へ、なめらかに変形する画像エフェクト技術のことです。

☆　メンバ関数

void SetDiv(int nSizeX,int nSizeY,int nXDiv,int nYDiv);

　モーフィングを行なう領域のサイズと、その分割数(10,10ぐらいで良い)を決めます。Calcを呼び出す前に呼び出してください。

void Set(int nSrcX1,int nSrcY1,int nSrcX2,int nSrcY2);

　ソース１とソース２の対応点集合を与える。この関数を何度呼び出しても構わない。

void Release(void);

　Setした対応点集合を破棄する。

void Calc(void); // 計算するのだ＾＾

　実際に、どの点がどの点に対応するのかを計算する。少し時間がかかる（それでも1/10秒とかの単位だが）ので、モーフィング中には呼ばない設計にしてください。

void OnDraw(CDIB32* lpSecondary,CDIB32* lpSrc1,CDIB32* lpSrc2,CDIB32 *lpTemporary,int nPhase);

　実際にモーフィングさせるるlpSrc1からlpSrc2へモーフィング。nPhaseは0-256(0ならばlpSrc1の画像、256ならばlpSrc2の画像。1から255ならば、その中間の画像),lpTemporaryはSecondaryと同サイズ確保してください。
