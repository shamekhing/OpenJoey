class CVolumeFader　【コンポーネント】

☆　内容

オーディオのミキシング用クラスCAudioMixerを利用して、MIDI、CD、WAVEに対して擬似的なフェードイン／フェードアウトを実現する。CDに対するフェードイン等は、技術的にも稀である。

CAudioMixerも参照すること。

☆　メンバ関数

LRESULT ResetVolume(eAudioMix e); // 初期ボリューム値に戻す
LRESULT FadeOut(eAudioMix e,int dwTime); // フェードアウトを開始させる
LRESULT FadeIn(eAudioMix e,int dwTime); // フェードインを開始させる
bool IsFade(eAudioMix e)const; // フェード中なのか？

dwTime[ms]後にボリュームのフェードは完了する。
eAudioMixについてはCAudioMixerクラスを参照のこと。

☆　注意点

フェードは、CAppIntervalTimerによるが、これはCAppBase::IsThreadValidでWindowMessageを処理しない限りは、コールバックされない。よって、CVolumeFaderのFadeOut/FadeInを呼び出したあとは、Sleepしては意味がない。CAppFrame::MesSleepを使うか、CAppFrame::IsThreadValidを呼び出す描画ループのなかで使うかである。
