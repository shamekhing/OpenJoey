class CBumpMap

☆　内容

CDIB32でBumpMapを行なう関数CBumpMapFastBlt,CBumpMapBltのために使うためのクラスです。

このクラスを使えば、うずまき、部分拡大、凸レンズ等、歪んだ転送が出来ます。

BumpMapを定義する構造体は、以下のものを使います。

struct SBumpMapChip { // 変移量
LONG x;
LONG y;
SBumpMapChip() { x = y = 0; }
};

この構造体で、転送元から転送先への歪み（変移量＝オフセット）を指定します。各ピクセルごとに、この歪みを指定するので、転送サイズ分のSBumpMapChip構造体が必要になります。この構造体を管理するクラスが、CBumpMapです。

☆　メンバ関数

void Resize(int x,int y); // 変移量をリサイズ(Reset兼ねる)する
void GetSize(int&x,int&y); // Resizeしたサイズを得る
SBumpMapChip* GetTable(void); // 変移量の先頭ポインタを得る
SBumpMapChip** GetStartTable(void); // bump tableの各ラインの先頭アドレステーブルを得る
