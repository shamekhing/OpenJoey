class CCPUID

☆　内容

CPUの判別を行ない、値を返します。PlainPentium,MMX Pentium,Pentium Pro以降(Celeron,Pentium II以降もこれに含まれる)の３種類を判別します。

☆　メンバ関数

static int GetID(void);

　CPU判別を行ないます。486以前ならば1,PlainPentiumならば2,MMX Pentiumならば3,Pentium Pro以降ならば4が返ります。
　今後、判別するCPUの数を増やすかも知れませんので、PentiumPro以降をチェックするには、

　　if (CCPUID::GetID() == 4)

ではなくて

　　if (CCPUID::GetID() >= 4)

と比較するようにしてください。
