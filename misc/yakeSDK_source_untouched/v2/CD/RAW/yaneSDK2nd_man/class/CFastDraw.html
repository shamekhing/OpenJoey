class CFastDraw 【コンポーネント】

☆　内容

このクラスは、CDirectDrawとほぼコンパチ。CDirectDrawに比較すると半透明等が高速であり、加色合成、減色合成、定数倍加色、定数倍減色等をサポートする。全体的にCDIBDrawよりも高速。今後は、描画はこのクラスをメインに使うと良いと思われる。

CDirectDrawをCFastDrawに差し替えて、CPlaneをCFastPlaneとすれば、コンパイルは通るはず。

CDirectDrawと異なり、CFastDraw自体は、CPlaneBaseの派生クラスでは無いので、セカンダリに対して描画を行なう場合は、

GetSecondary( )でセカンダリを取得して、そこに描画すると良い。
