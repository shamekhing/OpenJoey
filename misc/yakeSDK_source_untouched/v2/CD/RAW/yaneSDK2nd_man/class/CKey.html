class CKey,CKey2,CKey3,CKey4 【コンポーネント】

☆　内容

簡易キー入力クラス。CVirtualKeyより派生。

CKeyBaseの派生クラスを利用して、CVirtualKeyで、それらをひとまとめにして管理している。

☆　メンバ関数

CKey,CKey2,CKey3,CKey4は、キーデバイス登録済み、仮想キー設定済みのクラスです。キーの調査には、CVirtualKeyの

bool IsVKeyPress(int vkey); // 仮想キーを調べる
bool IsVKeyPush(int vkey); // 仮想キーを調べる

この２つの関数を使ってください。（入力は、Inputメンバ関数を使うこと）

☆　CKeyの仮想キー割り当て

０：ＥＳＣキー
１：テンキー８，↑キー，ジョイスティック↑
２：テンキー２，↓キー，ジョイスティック↓
３：テンキー４，←キー，ジョイスティック←
４：テンキー２，→キー，ジョイスティック→
５：スペースキー，ジョイスティック　ボタン１
６：テンキーEnter,リターンキー，左シフト，右シフト。ジョイスティック　ボタン２

☆　CKey2の仮想キー割り当て

０：ＥＳＣキー
１：テンキー８，↑キー，ジョイスティック↑
２：テンキー２，↓キー，ジョイスティック↓
３：テンキー４，←キー，ジョイスティック←
４：テンキー２，→キー，ジョイスティック→
５：スペースキー，Ｚキー，ジョイスティック　ボタン１
６：テンキーEnter,リターンキー，Ｘキー，ジョイスティック　ボタン２
７：Ｃキー，ジョイスティック　ボタン３
８：Ａキー，ジョイスティック　ボタン４
９：Ｓキー，ジョイスティック　ボタン５
１０：Ｄキー，ジョイスティック　ボタン６

☆　CKey3,4の仮想キー割り当て

CKey3は、CKeyからジョイスティックの割り当てを除いたものです。
CKey4は、CKey2からジョイスティックの割り当てを除いたものです。
