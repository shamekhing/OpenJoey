class CFlagDebugger

☆　内容

ゲーム等の実行時に配列の内容を視覚的に確認し、マウスでエディットするためのクラス。

CHDCLayer派生クラスなので、このクラスをnewした瞬間から有効になります。

☆　メンバ関数

void SetMouse(CMouseEx*lp);

CMouseExのポインタを渡しておきます。呼び出し側で、毎フレームごとにCMouseEx::Flush( )が呼び出されていなければなりません。

void SetFlag(DWORD*lpdw,int nSize);

エディットしたい配列の先頭アドレスと、その配列の要素数を設定します。


上記の２つの関数で初期化さえしておけば、あとは、CDIBDraw::OnDraw（CDirectDraw::OnDraw）のときにコールバックされます。

☆　操作方法

エディット画面の操作は、マウスのみです。左で決定。右で５１２要素次に送ります。○はゼロ、●は非ゼロ、☆は現在編集中の配列を示します。現在編集中の配列は、画面上部にビット表記（３２ビットなので３２個の○と●）で表現されています。編集は、この部分に対して、マウスの左クリックで直接ビットをＯＮ／ＯＦＦするか、下の＋１，＋１０，＋１００，−１，−１０，−１００を左クリックして、まとめて値を加減するかです。
