使うと便利なマクロ集(YTL/yaneMacro.hで定義されています)

WARNINGマクロ

☆　内容

デバッグ用の診断マクロです。C言語のASSERTに近いものです。

☆　使用例

　　WARNING(lp==NULL,"lp==NULLだ！！");

のようにしておけば、デバッグビルドだとそこで停止します。

（故意にメモリエラーを出すコードが含まれているので、そのままデバッグモードに入ることが出来ます）

ZEROマクロ

☆　内容

ある領域をゼロで埋めるためのマクロです。

☆　使用例

　　int x[100];
　　ZERO(x); // これで、xの要素すべてが0でクリアされます。

DELETE_SAFEマクロ

☆　内容

ポインタが非NULLならばdeleteを掛けます。その後、ポインタにはNULLが代入されます。

// ↑これ使わずにauto_ptrを使いましょうね＾＾；

☆　使用例

　　DELETE_SAFE(m_lpBox);

DELETEPTR_SAFEマクロ

☆　内容

配列へのポインタが非NULLならばdeleteを掛けます。その後、ポインタにはNULLが代入されます。

// ↑これ使わずにauto_array("YTL/auto_array.h"のなかにある)を使いましょうね＾＾；

☆　使用例

　　DELETEPTR_SAFE(m_lpaBox);

ｅｆマクロ

☆　内容

else ifの簡略表現です。

☆　使用例

　　if (x>=10) { ... }
　　ef (x=<0) { ... }

TRACE　マクロ

☆　内容

トレース用のマクロ:ファイル名とラインナンバーをError.logに出力する

NELEMS(array) 　マクロ

⇒　(sizeof(array) / sizeof(array[0]))

☆　内容

配列の要素数を返すマクロ( Number of Elements)　これを使えば、マジックナンバーを少し減らすことが出来る＾＾；　カーニハンの「プログラミング作法」で紹介されていたもの。
