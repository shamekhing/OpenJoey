
このフォルダにあるのは、YTL(Yaneurao Template Library)

	auto_ptrEx			:	auto_ptr拡張バージョン。
	auto_array			:	auto_ptrの配列バージョン。
	auto_vector_ptr		:	auto_ptrのvectorを使いやすくしたもの。


	vectorEx			:	vectorの拡張バージョン + chain
	auto_ptrDx			:	auto_ptrEx + FactoryMethod + PrototypePattern w/z auto_ptrEx。すげー＾＾；

	smart_ptr			:	smartポインタ。(Java的な)
	smart_array			:	smartポインタの配列バージョン。
	smart_vector_ptr	:	smartポインタのstd::vectorバージョン
		→　やねうらおＨＰ「天才ゲームプログラマ養成ギプス　第１１章」参照のこと

	swap				:	値を交換するだけの関数＾＾；

	factory				:	FactoryMethod + PrototypePattern w/z auto_ptrEx
		→　やねうらおＨＰ「天才ゲームプログラマ養成ギプス　第６章」参照のこと

	surrogate_mother	:	試しに作ってみた代理母クラス。使いものにならない＾＾；

	function_callbacker	:	クラスメンバ関数へのコールバックを実現する
		→　やねうらおＨＰ「天才ゲームプログラマ養成ギプス　第１０章」参照のこと

	mediator			:	（Java的な）内部クラスを仮想的に実現する
		→　やねうらおＨＰ「天才ゲームプログラマ養成ギプス　第１１章」参照のこと

	yaneMacro			:	簡単なマクロ集
