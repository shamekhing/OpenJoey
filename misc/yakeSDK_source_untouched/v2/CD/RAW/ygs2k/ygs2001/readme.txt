
	ygs2001 ( ygs2k for Visual Novel ) version 1.11

											programmed by やねうらお '01/11/25
														lastupdate '02/04/15

・フォルダ編成

	ygs2001.exe		：実行ファイル
	readme.txt		：このファイル
	変更点.txt		:ygs2kからの変更箇所
	+manual			:ygs2kのマニュアル一式
	+script			:スクリプトを入れるフォルダ
		このなかにgamestart.cというファイルが入っているが、
		これをテキストエディタで書き換えて使う
	+source			:ygs2001(version 1.11)のプログラムソース。
		ygs2001を使うだけの人には必要無い。
	+grp			:ノベル用のグラフィック素材の入ったフォルダ
		cursor.bmp		:	カーソル素材
		+kari			:	仮素材（これは著作権物なので転載／流用禁止）
		+scenario		:	シナリオ画面のための必要素材。
					これは自由に変更して流用して構わない。
					ボタン素材は、私がテキトーに作ったやつなので
					ゴミまじってますが、このへんは適宜差し替えてください。

・マニュアルのほうは、スクリプトマニュアルに書いてありますが、
　ygs2kからの変更点がありますので、それについては、
「変更点.txt」をご覧ください。

・このバージョンは、ygs2kで弱かったブレンド機能・加色合成・減色合成の
	機能を強化してあります。

・このバージョンは、ビジュアルノベル用に特化された命令が用意してあります。
	（詳しくは、私のお楽しみＣＤのほうの原稿をご覧ください）

・このビジュアルノベルエディションでは、ビジュアルノベル用の素材等が添付されて
　います。このグラフィック素材自体は、誰でも自由に流用できます。
	（ただし、立ちキャラ素材grp/kari/01.ygaおよび、背景画像grp/kari/01.jpgは
　　　著作権物なので、２次利用はご遠慮くださいませ）

・ソースコードをコンパイルするには、sourceフォルダの中身以外に
	Visual C++6.0およびyaneSDK2nd 1.54以降が必要です。
（ygs2001を使用するのには、このコンパイル作業は必要ありません。
　参考として添付してあるとご理解ください）

・実行ファイルは、実行すると、自動的にscript/gamestart.cという
　スクリプトを実行します。そのファイルが無ければ、そのまま終了します。
　その他は、ygs2kのサンプル等を見てください。

・ソース・実行ファイルについては、自由に引用・再配布して使って構いません。
　うるさいことは一切言いません。

　それでは良きプログラミングライフを！
