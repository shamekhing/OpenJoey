《このバージョンの説明》

	layout_defineは完璧ではありませんが、座標の指定は出来ます。
	そちらで変更していただきたいのは、上記の設定ファイルです。
	htmlの基本タグ、禁則処理は行なっています。
	文字表示は、１８文字×３行（禁則文字の含まれる行は１９文字）は実現している
	ので、表示レイアウトの確認はしていただけるかと思います。

	☆　ファイル名の登録方法

	０番から順番にファイル名を登録していきます。
	シナリオファイル側からは、この番号で呼び出します。
	ただし、ネームプレートに関しては、番号ではなく、シナリオの会話部分の
	先頭文字で自動判別します。（そのため、name_defineだけ少し記述方法が
	違います。詳しくは、添付のcfg/name_define.txtをご覧ください）

	また、変更した設定ファイルは必ずバックアップを取っておいてください
　　　　　　　　　　　　　　　　~~~~~~~~~~~~~~~~
	新しいバージョンを上書きしたときに、誤って消してしまわないように
	するためです。

	☆　ｙｇａ画像について

	アンチェリの必要な画像は、YGA形式を使用します。
	ネームプレート、立ちキャラ、フェイスマークにYGA形式を使います
